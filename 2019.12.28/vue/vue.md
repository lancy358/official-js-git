### Vue
    用的是 MVVM框架  Model View  View Model
    原生用的是MVC框架 M-model数据，V-view视图  C-控制器control
    难点：父子组件的传递,路由配置和使用,生命周期,vuex,**逻辑**,做项目
### 使用 Vue
    -引vue.js 文件
    - 在html中创建一个根元素  <div id="box"> </div> 
    -实例化vue 
        let v=new Vue({})
    -配置参数
        -el:"#box" el 不能变只能写成el
        -data（存数据）：在 new Vue 下值为 对象；
    -输出数据用 双花括号{{放data的key值即可}}（小胡子）
